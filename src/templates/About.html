About Us


VoyageRoo is a voyagerific application developed by a set of voyagerously skilled developers from team VOYAGER. VoyageRoo lets you pick your favourite movies, and/or vote for current favorites, with the goal of facilitating a voyagerlicious movie night experience with your loved ones.

The app was built using AngularJS, Angular-Material, Bootstrap, Bower, Gulp, Firebase, with a MySQL database hosted on AWS communicating using Spring REST endpoints. VoyageRoo uses the TMDb API but is not endorsed or certified by TMDb.
